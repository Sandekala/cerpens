# Cerpens

Hai, ini adalah website **Cerpens** tempat dimana anda dapat menulis cerpen anda sendiri :)

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
